# dn_position

## What the mod does:
It shows coordinates (x, y & z), angle & symbol for the direction of the world (North - N, South - S etc.)

## Version:
0.1alpha

## Installation:
Unpack into mods' folder.

## Dependencies:
default (found in minetest_game)
